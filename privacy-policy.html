Privacy Policy

Last updated: July 19, 2026

Maime Content Gen ("the app") is a single-user desktop application for macOS. It is built and used by an individual creator to turn their own music tracks into video content. This policy explains what data the app handles and how.

What the app stores

All project data — audio files, transcribed lyrics, generated video clips, and rendered exports — is stored locally on the user's Mac, inside the project folder the user chooses. Nothing is uploaded to any server operated by the app's developer, because no such server exists.

API keys and access tokens

API keys for third-party services (video generation, language models) and OAuth access/refresh tokens for YouTube and TikTok are stored exclusively in the macOS Keychain, protected by the operating system. These credentials are used only to make requests directly from the app to the respective service (OpenAI, Anthropic, the video generation provider, YouTube, TikTok) on the user's behalf, and are never transmitted anywhere else.

What gets sent to third parties

When the user explicitly triggers an action, the app sends data directly to the relevant third-party API over an encrypted connection:

  • Audio or lyric text to the configured language model provider, to plan a storyboard.
  • Generation prompts to the configured video generation provider, to generate video clips.
  • The finished video file to YouTube and/or TikTok, only when the user chooses to publish it, and only to the account the user has authorized via that platform's own sign-in flow.

The app does not use analytics, tracking, or advertising SDKs of any kind, and does not share data with any party other than the third-party services the user has explicitly configured.

Deleting your data

Since all data lives on the user's own Mac, it can be deleted at any time by removing the project folder or using the app's built-in reset option. Revoking API keys or OAuth access can be done at any time from the respective provider's account settings.
